### 概览

欢迎来到《鄞年・思叙：鄞州中学校园期刊》（*Yinzhou High School Campus Journal*）创刊号的官方 GitHub 仓库，我们将会把技术范围内能上传／能归档的所有文件都上传到此处。

### 为什么要建立此仓库？

我们原先用 Craft 跑通了全流程，但为了下期更大规模的编辑部的顺利运转，也为了将我们的幕后工作流程（如未刊发的稿件）展现给各位，特此建立该仓库。
